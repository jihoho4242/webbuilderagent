# Agent OS P5 Release Evidence Contract

A P5 release evidence bundle is the only allowed basis for claiming v3.2 Agent OS readiness. It must include constitution hash, policy coverage, tool gateway coverage, HITL v2 evidence, replay consistency, red-team results, eval results, Brain audit, self-improvement dry-run evidence, script-executor neutralization audit, validation command results, and blockers.

If any required evidence is missing or failed, `release_ready` must be false.
